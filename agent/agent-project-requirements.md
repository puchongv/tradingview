## üìö Metabase SQL Library (Added 2025-09-14)

### Parameters (conventions)
- `interval_target` (Text): 10 | 30 | 60 (accepts 10m/30m/60m)
- `ex_hours` (Text, CSV): hours 0‚Äì23 to exclude
- `ex_days` (Text, CSV): DOW 0=Sun..6=Sat to exclude
- `target_dow` (Text): ALL | 0..6
- `target_hour` (Text): ALL | 0..23
- `payout` (Number): default 0.80
- `investment` (Number): default 250
- `min_trades` (Number): default 1‚Äì3 depending on chart

### Provided Queries
- Winrate pivot (rows: hour, cols: Sun‚ÄìSat) ‚Äì no TZ
- By-hour winrate (direct AVG) ‚Äì cross-check
- Bar chart daily PnL per `strategy | action` (series_key) ‚Äì noise-safe
- Breakdown per DOW√óHour listing strategies with: `total_trades`, `wins`, `losses`, `win_rate`, `pnl`, `bucket_total_trades`, `bucket_win_rate`, `share_pct`

### Formulas
- Winrate: `AVG((result='WIN')::int) * 100`
- PnL: `(wins * payout - losses) * investment`
- Share%: `100 * strategy_total / bucket_total_trades`
